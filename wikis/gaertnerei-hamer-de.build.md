# Build decision log — Gärtnerei Hamer

Generated by `/create`, then redesigned to a **Botanical Editorial** direction via the **ui-ux-pro-max** skill, with real **fal.ai genmedia** personalization and a much more image-led layout.
Stack: **Next.js 16.2.9 · React 19 · Tailwind v4** (App Router). German-first, token-driven, multi-page (8 routes, all prerendered static).

## Design direction — Botanical Editorial
Asymmetric magazine layout: full-bleed image hero + bands, **bento gallery + lightbox**, oversized serif display, captions over paragraphs, parallax + blur-fade reveals, 1px image outlines, layered shadows, press-scale buttons.
- **Type:** **Playfair Display** (display) + **Figtree** (body) + **JetBrains Mono** (uppercase micro-labels) — all self-hosted via `next/font`.
- **Color:** botanical green tokens (AA-safe `secondary`/`muted`), warm paper/sage surfaces. No hardcoded values in components (verified).
- Reusable components: `ImageBand` (parallax), `Gallery`/lightbox (keyboard + focus), `Marquee`, `Parallax`, `Eyebrow`, `PageHeader` (full-bleed photo).

## Imagery — real photos + AI genmedia
- **~40 of the brand's own photos** pulled from the live site into `src/assets/brand/<category>/` (hero, nursery, floristik, wedding, funeral, plants, garten, grab, team) and used via `next/image`. They carry the site; copy was cut ~60–80%.
- **5 AI images** (fal.ai **Nano Banana 2**, committed to `src/assets/generated/`, regen via `scripts/generate-media.sh`):
  - `hero.jpg` — signature brand hero (text-to-image, 2K).
  - `showcase.jpg` — "bunt" botanical band (text-to-image, 2K).
  - `floristik/garten/plants` — real photos restyled via `fal-ai/nano-banana-2/edit` for a cohesive editorial grade. **People photos never edited.**
  - Models: `fal-ai/nano-banana-2` + `/edit`. Cost ≈ $0.56 total. Prompts in `scripts/generate-media.sh`.
