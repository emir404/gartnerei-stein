# Build decision log — Gärtnerei Stein

Generated by `/create` from `wikis/gaertnerei-stein-de.md`. The user asked to
**change all the layout** and **not repeat the previous (Gärtnerei Hamer) site**,
so this is a deliberately different build on every axis.

Stack: **Next.js 16.2.9 · React 19 · Tailwind v4** (App Router, Turbopack).
German-first, token-driven, **single static page**.

---

## How this diverges from the Hamer build (by request)

| Axis | Hamer (previous) | Stein (this build) |
|---|---|---|
| Structure | 8 routes, multi-page | **One static page**, anchored sections |
| Direction | Full-bleed photo magazine | **Type-led editorial "Botanischer Almanach"** |
| Signature device | Parallax image bands + bento lightbox gallery | **Sticky numbered "spine" index** with scroll-spy + a growing progress line |
| Imagery | ~40 real photos carrying the layout | **No photo gallery** — one AI showcase image + hand-drawn **SVG botanical motifs** + token color-fields |
| Type | Playfair Display + Figtree + JetBrains Mono | **Fraunces (+italic) + Mulish**, tracked-caps labels (no mono) |
| Color | Evergreen + honey-gold on warm paper | **Garden green + peony-rose on stone/oat paper** |
| Nav | Persistent call/email + fixed mobile contact bar | Condensing top bar + **mobile dropdown**; the spine is the wayfinding |
| Motion | Ken-Burns, marquee, parallax | Restrained **reveal-on-scroll** + the spine fill only |

> Note: another company's real photos that shipped with the Hamer build
> (`src/assets/brand/**`, `src/assets/generated/**`) and all Hamer routes/components
> were **deleted**, not reused.

## Page structure (sections + why)

1. **Hero (`#start`)** — type-forward. Eyebrow heritage line, oversized Fraunces
   headline *"Aus einer Hand: gewachsen & gebunden."* (nursery = grown, florist =
   bound), lede, CTAs (Sortiment + Anrufen), and a token-driven **"almanac card"**
   that teases the two locations — no photo needed.
2. **Lede** — a slim editorial family statement (Grußwort). Sets voice early.
3. **Sortiment (`#sortiment`)** — the **almanac index**: big tabular numerals in
   the gutter, hairline-ruled rows, staggered reveal. Five offerings drawn from
   the profile.
4. **Zwei Orte (`#orte`)** — the brand's unique hook (nursery in Groß Steinrade +
   florist at famila Stockelsdorf) as a two-card split on a stone band, each with
   full address, an hours table, phone + route.
5. **Über uns (`#ueber-uns`)** — heritage statement + an honest 3-stat band
   (1976 / 2 Standorte / Familie). No invented awards or ratings.
6. **Showcase** — full-bleed AI botanical band (see below) with an overlaid
   caption.
7. **Kontakt (`#kontakt`)** — inverted deep-forest CTA band: "no online shop —
   two places to visit," both phone numbers, addresses, hours, Facebook.
8. **Footer** — wordmark, nav, both locations, © Gärtnerei Stein GmbH.

Cut vs. a generic SMB template: no contact form (the business lists **no email** —
phone + visit + maps are the honest CTAs), no blog, no team page (no verified team
data beyond "the Stein family").

## Copy choices (key)

- Voice: warm, local, family-run, plainspoken German (profile `voice.tone`).
- Headline pairs the two halves of the business: *gewachsen* (Gärtnerei) &
  *gebunden* (Blumenladen).
- Only profile-supported claims: **since 1976**, family business, **two
  locations**, balcony/bed plants + cut flowers + floristry, personal advice,
  fair prices. **Omitted** the unverified ~4.7★/73-review figure and any "eigene
  Anzucht" propagation claim (softened to "aus unserer Gärtnerei").
- `brand.tagline` was `null` → no fake slogan invented; the hero headline + the
  showcase caption carry the brand line instead.

## Tokens used (+ derived/refined values logged)

Source: the wiki's Design Tokens block. Wired in `globals.css` (raw `:root` →
`@theme inline` colors/fonts → `@theme` structural) and `layout.tsx` (fonts).

- **Explicit from wiki:** primary `#2b5740`, secondary `#4f6b43`, accent
  `#c25a66`, background `#f5f1e8`, surface `#eae3d3`, text `#23251d`,
  border `#e3dccb`; radius `6/12/20`; shadow set; type Fraunces/Mulish; scale
  16px·1.25; voice/imagery.
- **Palette-sourced support tokens** (all present in the wiki `palette`, not
  invented): `--primary-deep #173a2a` (inverted bands/footer), `--accent-soft
  #ebc9c2` (blush), `--card #ffffff`, `--ink #181a12`.
- **Derived (logged):**
  - `logo.*` = all `null` → **generated Fraunces wordmark** ("Gärtnerei Stein")
    in header/footer (per DESIGN_TOKENS derivation).
  - `muted` refined `#6b6b5b` → **`#5e5e4d`** to clear WCAG AA (≥4.5:1) for small
    secondary text on both paper and stone.
  - `voice.formality` = neutral; `imagery.mood` used for the showcase prompt.
- **Custom utilities** (token-only, no raw values): `.label`, `.nums`,
  `.bloom-field` / `.bloom-field-deep` (palette mesh), `.grain`, `.reveal`,
  `.rule`. Reveal + scroll motion are disabled under `prefers-reduced-motion`.

**Token audit:** grep of `src/components/**` and `src/app/**` for hex / rgba /
font-family / inline radius-shadow → **0 hits** outside the `:root` token block.

## Brand showcase

**Decision object**
```json
{
  "type": "text-to-image",
  "placement": "section-break",
  "concept": "An abundant hand-tied seasonal bouquet beside potted balcony plants on a weathered pale-limestone surface in a calm sunlit florist's workshop",
  "constraints": {
    "aspect": "16:9",
    "noText": true,
    "palette": ["#2b5740", "#f5f1e8", "#eae3d3", "#4f6b43", "#c25a66"],
    "mood": "warm, fresh, local, family, grounded, seasonal",
    "avoid": ["text", "logos", "off-palette colors", "generic stock", "people", "cold/sterile tones"]
  },
  "rationale": "Unites nursery (potted plants, terracotta) + florist (bouquet) + the name 'Stein' (stone/limestone), in-palette, with calm negative space on the left for the overlaid caption."
}
```

**Model:** `fal-ai/nano-banana-2` (text-to-image, 2K, jpeg) via the **`genmedia`**
CLI — generated once, downloaded to `src/assets/generated/showcase.jpg`, committed,
served through `next/image` (optimized + blur placeholder). Reproduce via
`bash scripts/generate-media.sh`.

**Exact prompt**
> An abundant hand-tied seasonal bouquet of garden flowers beside a few potted
> balcony plants, arranged on a weathered pale limestone and stone surface in a
> calm sunlit florist's workshop. Wide cinematic framing with generous calm
> negative space on the left for overlaid text. Editorial botanical still-life
> photography, soft shallow depth of field. Warm, fresh, grounded, family-run,
> seasonal mood. Color palette limited to deep garden green #2b5740, warm oat
> cream #f5f1e8, soft stone grey-beige #eae3d3, sage green #4f6b43 and a single
> peony-rose accent #c25a66. Warm natural North-German daylight, high detail,
> photoreal magazine quality. **No text, no words, no letters, no watermark, no
> logo, no people, no hands, no faces.** *(negative cues baked into the prompt
> tail; Nano Banana 2 takes no separate negative_prompt)*

**Quality gate:** **passed on the first generation** — subject on-concept
(bouquet + potted plants + limestone), palette in tolerance (greens/oat/stone +
one rose peony, warm terracotta pots read as stone/clay neutral), **no text, no
people**, and **left negative space** present for the caption. No regenerate, no
fallback needed. Asset: `src/assets/generated/showcase.jpg` (2752×1536).
The CSS `.bloom-field` fallback remains wired in `Showcase.tsx` (set `showcase =
null` to use it) for a zero-asset rebuild.

## Revision — real photos, clipped logo, dash cleanup (2026-06-29)

User-requested follow-up: *"put a few more images, drop the long dashes, and
use their logo (clip its background)."* Five owner-supplied photos (`images/`)
and the real wordmark (`logo.jpg`) were brought in.

**Real logo (background clipped via genmedia).** `logo.jpg` (green "Gärtnerei
Stein" wordmark on white) was uploaded to fal and run through **BiRefNet v2**
(`fal-ai/birefnet/v2`, `output_format=png`) to drop the white background →
`src/assets/brand/logo.png` (transparent, 200×200). It now replaces the Fraunces
text wordmark in **SiteHeader** (44–48px, beside the italic "seit 1976") and
**SiteFooter** (64px). Reproducible via `scripts/generate-media.sh` (new
`clip_logo` step; `geturl` extended to read birefnet's singular `result.image`).

**Photos (5), placed editorially — not a generic gallery.** Copied to
`src/assets/photos/`, served through `next/image` (static import → optimization +
blur-up), framed in brand tokens with a restrained hover-zoom via a new shared
**`Figure`** component (overlay-caption variant reuses the showcase treatment):
- **Hero** — `gewaechshaus.jpg` (greenhouse interior, portrait) replaces the
  token-only "almanac card" as the signature hero visual.
- **Sortiment** — `geranien.jpg` as a wide cinematic band under the header.
- **Zwei Orte** — each location card gains a photo header: `stiefmuetterchen.jpg`
  (Gärtnerei) and `chrysanthemen.jpg` (Blumenladen). The decorative `Sprig` was
  dropped from these cards (the photo now carries the visual).
- **Über uns** — `schild.jpg` (the painted storefront sign). Its third-party
  "© LUT" watermark is cropped out purely in-layout via `object-cover object-top`
  in a 16/10 frame (keeps the top ~83%), so the source file is left untouched.

Lede and Kontakt stay type-only for rhythm; the AI `showcase.jpg` is unchanged.
Photo count on the page: 8 `<img>` (5 photos + logo ×2 + showcase).

**Em dashes dropped.** Every visible "—" in copy was rewritten to flow with
commas/colons/periods (e.g. hero, lede, sortiment items, über-uns, kontakt,
showcase caption, page/OG titles, JSON-LD `name`); numbered section labels now
read `02 · Sortiment` etc. The en-dash **ranges** in opening hours (`Mo – Fr`,
`08:00 – 18:00`) are correct range notation and were intentionally kept. Code
comments (not rendered) keep the house em-dash style. Served-HTML grep: **0** em
dashes.

**Verification:** `bun run build` ✓ clean (TypeScript + static prerender).
Served-HTML smoke test ✓ — all 5 photo `alt`s, both logo instances, and the
showcase present; zero "—" in the visible output.
